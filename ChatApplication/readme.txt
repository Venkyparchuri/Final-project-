Adapter Design Pattern - 
I have created Adapter design pattern where SingleChat and GroupChatStatus are the Interfaces. My main class is ChatAdapterMain.

Command Design Pattern - 
I have created Command Design Pattern where NewContact is the interface, SendMessage and ReceiveMessage is a concrete class implements NewContact. The ChatCommandMain is the main class. 

Facade Design Pattern - 
I have created ChatFacadeMain as the main class, GroupChat as interface, Receiver, sender and receiver as concrete classes implementing the same interface and ChatServer as Facade class.

Factory Design Pattern - 
I have created ChatFactoryMain as the main class, GroupChat as interface, Receiver, sender and receiver as concrete classes implementing the same interface and ChatServer as factory class to generate object of concrete class.
